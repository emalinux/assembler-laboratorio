# Assembler Computer – Sito Laboratorio

Sito ufficiale del laboratorio informatico Assembler Computer.

Progetto statico sviluppato con **Hugo Extended**, layout completamente personalizzato, architettura CSS modulare e ottimizzazione SEO di base.


[![Netlify Status](https://api.netlify.com/api/v1/badges/9a8969f0-34e6-4361-b00f-501f1f2b818f/deploy-status)](https://app.netlify.com/projects/assembler-laboratorio/deploys)

---

## 🔧 Stack Tecnologico

- Hugo v0.147.9 (extended)
- HTML5 semantico
- CSS custom (design system interno)
- JavaScript vanilla
- Deploy su Netlify
- Versionamento GitHub

---

Realizzare un sito:
